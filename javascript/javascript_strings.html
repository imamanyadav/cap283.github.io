    JavaScript Strings

    JavaScript strings are used for storing and manipulating text.

    A JavaScript string simply stores a series of characters like "Vetri Pandi".

    A string can be any text inside quotes. You can use single or double quotes.

     

    Example

    var Google = "Larry Page"
    var Apple = 'Steve Jobs'

    Special Characters

    The backslash escape character turns special characters into string characters.

    Because strings must be written within quotes, JavaScript will misunderstand this string.

     

    Example

    var x = 'It\'s alright';
    var y = "We are the so-called \"Vikings\" from the north."

    Strings Can be Objects

    Normally, JavaScript strings are primitive values, created from literals: var Name = "vetri"

    But strings can also be defined as objects with the keyword new: var Name = new String("John")

     

    Example

    var x = "vetri"
    var y = new String("vetri");
